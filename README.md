# Usage 

Go to the actions tab on this github repo. 
Download the artifact for your machine.

<img width="798" height="796" alt="Screenshot 2026-05-10 at 23 26 03" src="https://github.com/user-attachments/assets/0904aac5-8372-4a1d-a420-58a16b9a7b6c" />

You may get a security warning about this app. On windows you may need to whitelist the app in f secure. 
Pick an image. 
The characters in the image will automatically be extracted and identified. 
Be sure to click on each and ensure the cropping is good and nothing is incorrectly assigned. 
Optionally upload more images to fill in any missing characters.
Optionally turn on 'Super Resolution' In options to average the first 10 images of a character.


Click save and [enjoy](https://fontsee.com/) the font!.
